### Objective

Analyze the Sample Superstore dataset using SQL and perform business-oriented analysis through filtering, aggregation, sorting, and reporting queries.

### Concepts Covered

* Data Exploration
* WHERE Filters
* GROUP BY Aggregations
* Sorting and Limiting Results
* Business Analysis Queries
* Data Validation
* Sales & Profit Analysis
